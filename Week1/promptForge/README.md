# PromptForge

PromptForge is a Gradio-based AI application that demonstrates prompt engineering concepts using multiple AI personas.

## Features

- Mentor persona
- Code Reviewer persona
- Few-shot prompting
- Streaming responses
- JSON parsing for structured code reviews

## Tech Stack

- Python
- Gradio
- Groq API
- Llama 3.3 70B

## Run

```bash
python app.py
